# 📊 Customer Churn Prediction using Machine Learning

## 💼 Business Problem
Companies lose millions due to customer churn. This project predicts high-risk customers using Machine Learning and provides actionable business insights for customer retention.

---

## 🛠️ Technologies & Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
* **Model:** Random Forest Classifier
* **Performance:** **95.06% Accuracy**
